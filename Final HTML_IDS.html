Conclusion:¶

The accuracy score for our Regression model is 48%, which is clearly not what we expected. The main reason for this, is that the predicted variables didn't quite explain the dependent variable, which is Transfer fee. R square value is 72%, so, 72% of variance is explained by our model. We tried different values of K. When the K value is 25 the results are optimal. The overall accuracy of KNN model is 71.6%, which is not that ideal. So, the final conclusion is: If the predictor variables would have been different we could have explained the Transfer fee for the player more efficiently.
